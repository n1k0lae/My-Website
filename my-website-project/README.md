# My Website Project

This project is a personal website designed to showcase my experience in customer service, highlight my projects, and provide a platform for customer stories and testimonials. 

## Project Structure

The website consists of the following main components:

- **index.html**: The main entry point of the website, linking to all other pages.
- **pages/**: Contains individual HTML files for different sections of the website:
  - **about-me.html**: A detailed description of my background and skills.
  - **projects.html**: An overview of my projects with links to detailed pages.
  - **customer-stories.html**: Testimonials and stories from customers.
  - **resume.html**: My professional resume.
  - **contact.html**: A contact form and my contact information.
  - **projects/**: A subdirectory containing detailed pages for each project:
    - **adaptive-customer-support-ai-system.html**: Details about the Adaptive Customer Support AI System project.
    - **customer-experience-framework.html**: Information on the Customer Experience Framework project.
    - **soul-screenshots.html**: Insights into the Soul Screenshots project.
  
- **assets/**: Contains all static assets for the website:
  - **css/**: Contains the stylesheets for the website.
    - **styles.css**: The main CSS file for styling the website.
  - **js/**: Contains JavaScript files for interactivity.
    - **main.js**: The main JavaScript file for dynamic features.
  - **images/**: A directory for images used throughout the website.

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the `index.html` file in your web browser to view the website.
3. Modify the content in the `pages/` directory to personalize your website.
4. Update styles in `assets/css/styles.css` as needed.
5. Add any additional images to the `assets/images/` directory.

## Project Goals

- To create a professional online presence that showcases my skills and experiences in customer service.
- To provide potential employers and clients with easy access to my projects and customer testimonials.
- To facilitate communication through the contact page.
